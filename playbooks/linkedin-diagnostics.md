# LinkedIn Diagnostics — When Metrics Are Low

*Symptom → root cause → fix. Check the first row of each section first (highest impact).*

## Low Accept Rate (<35%)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| Headline = job title, not value prop | Rewrite: "[Outcome] for [audience]" | ★★★★★ |
| Wrong ICP targeting | Tighten: industry + seniority + company size | ★★★★★ |
| Low mutual connections | Connect adjacent people first for social proof | ★★★★☆ |
| Amateur profile photo | Professional headshot, natural, approachable | ★★★★☆ |
| Sending on weekends | Shift to Tues–Thurs | ★★☆☆☆ |
| "I don't know this person" reports | Add personalised CR notes OR tighten ICP | ★★★☆☆ |

## Low Intro Reply Rate (<8%)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| Pitch on intro instead of conversation | Lead with observation/question, not offer | ★★★★★ |
| Messaging 24hr+ after accept | Automate within 1hr of accept | ★★★★★ |
| Generic "great to connect" opener | Reference their content/role change/mutual | ★★★★☆ |
| Message >150 words | Cut to 2–4 sentences, one question | ★★★☆☆ |
| Profile doesn't back up the intro | Audit what they see when they click your name | ★★★☆☆ |

## Low FU Reply Rate (<10% cumulative)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| All FUs repeat same ask | Different angle each: value, proof, scarcity, question | ★★★★★ |
| FU1 is templated | Hyper-personalise FU1 — highest-leverage FU | ★★★★★ |
| No passive touches between msgs | Profile view Day 2 + post like Day 5 | ★★★★☆ |
| Cadence too aggressive (<24hr) | Space 2–4 days apart minimum | ★★★☆☆ |
| No content on your profile | Post 2–3×/week, feed presence warms passively | ★★★☆☆ |

## Low Call Booking Rate (<15% of conversations)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| Vague CTA: "let's chat sometime" | Specific: "15-min cognitive audit — here's my calendar" | ★★★★★ |
| No Calendly link | One click to book, eliminate friction | ★★★★☆ |
| Asking too early (first reply) | Build 2–3 exchanges before the ask | ★★★☆☆ |
| No social proof in conversation | Drop case study naturally: "worked with someone similar…" | ★★★★☆ |
| Website undermines credibility | Audit site. Use Calendly + testimonials page if weak | ★★★☆☆ |

## Low Show Rate (<65%)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| No reminder sequence | 24hr + 1hr reminders. SMS + email. Restate value prop | ★★★★★ |
| Booked >5 days out | Offer slots within 48hrs only | ★★★★☆ |
| "Discovery call" framing | Reframe: "assessment" / "audit" with deliverable | ★★★☆☆ |
| Friction to join | Simple Zoom link in calendar invite, no passwords | ★★★☆☆ |

## Low Close Rate (<15%)

| Root Cause | Fix | Impact |
|------------|-----|--------|
| Pitching before diagnosing | Structure: 60% diagnosis, 20% prescription, 20% offer | ★★★★★ |
| ICP misfit (right convo, wrong person) | Tighten upstream targeting, qualify harder in chat | ★★★★★ |
| No urgency / cost of inaction | Quantify what staying the same costs them | ★★★★☆ |
| Price without anchoring | Anchor against cost of problem, not competitors | ★★★☆☆ |
| No clear next step on call | End with specific action + timeline | ★★★☆☆ |
