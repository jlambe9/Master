## Encoding Compensation (Until Stack Takes Effect)

Agents must, for first 8-12 weeks:
- **Session start:** Brief context summary — don't assume recall
- **Multi-session tasks:** Include previous session's ending state + notes
- **Restructured tasks:** Present restructured version, not original
- **Category history:** Include previous categorisations

As Jamie starts saying "I know, I remember" → reduce scaffolding. This is itself signal.
