### EXP-003: Restructuring as Primary Intervention
- **H:** Most A-category tasks are A-S (structural), and restructuring alone resolves >50% of aversion without pharmacological intervention.
- **Variables:** Restructuring attempts, category shifts (A → B/E)
- **Measurement:** Track restructuring events and resulting category changes in task-log.
- **Start:** When task log has sufficient A-task data

### EXP-004: Dextroamphetamine × Time-of-Day
- **H:** Dextroamphetamine efficacy on B-task initiation varies by time of day, with morning showing better initiation rates.
- **Variables:** Time of day, category, initiation success (Y/N), duration
- **Measurement:** B-task initiation rates morning vs afternoon.
- **Start:** When task log has sufficient B-task data
- **Status:** NOT YET STARTED

### EXP-005: Task Initiation vs Task Structuring
- **H:** Jamie's initiation failures are primarily structuring failures — tasks stall at initiation because goal clarity or decomposition was never completed, not because of motor/emotional initiation deficit.
- **Variables:** Pipeline failure stage, task structure quality (from Layer 0A)
- **Measurement:** Correlation between procedure specification and initiation success.
- **Start:** When task log has 20+ profiled executions
- **Status:** NOT YET STARTED
