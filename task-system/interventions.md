# Category Routing & Pattern Recognition

## 🟢 E — Epistemic
- **Action:** Execute.
- **Risk:** Time-hogging. Agent flags: "You've been in [E-task] for [X hours]. [B/A task] due [when]."

## 🟡 B — Boring
- **Hierarchy:** Eliminate → Automate → Delegate → Restructure → Time-box (25 min max)
- **Scheduling:** Afternoon. Never burn morning PFC on B-tasks.
- **Never** chain two B-blocks back-to-back.
- **Pharma note:** Guanfacine expected to reduce PFC energy cost.

## 🔴 A — Aversive
- **First question:** "Would restructuring change the feeling?"
- **A-S (yes):** Restructure. Re-categorise. Log the shift.
- **A-I (no):** "Is this core to goals?"
  - **Yes → Growth:** High-energy window. Body doubling, accountability, smallest first action. Rate emotion 1-10 before/after.
  - **No → Delegate/eliminate.** Don't spend emotional capital on non-compounding tasks.
- **Scheduling:** Morning or evening 6-9pm. Never stack two A-I in one day.

## 🟠 M — Misrouted
- **Action:** Reroute BEFORE executing.
- **Common reroutes for Jamie:**
  - Verbal encoding required → visual/diagrammatic capture, or AI holds verbal content
  - Reading-heavy → audio/discussion input, or AI summarisation
  - Written output from blank → template (fill-in, not generate)
  - Sequential/speed-dependent → untimed chunks with save points
- **After reroute:** Re-categorise (often becomes E or B).

## Pipeline Failure Points

| Stage | What Fails | Common Cause for Jamie |
|-------|-----------|------------------------|
| Goal clarity | Don't know "done" | Low specification + high entropy |
| Decomposition | Can't break into steps | High complexity + novel type |
| Sequencing | Don't know what order | Unclear dependencies |
| Planning | Can't commit to when/how | Effort overestimation + temporal delay |
| **Initiation** | Can't start | Motor impulse, emotional override, SEEKING deprivation |
| In-progress | Stalling mid-task | WM overload, encoding failure, energy depletion |
| Completion | Can't finish/ship | Exposure anxiety, perfectionism, identity threat |
| Context persistence | Can't build on it tomorrow | Encoding failure — work doesn't consolidate |

## Pattern Recognition — Agent Surfaces When:

- Same task hits A **3+ times** → structural or intrinsic? Delegate?
- Category shifts after restructuring → apply same restructure to similar tasks
- Prediction consistently wrong → update model, ask what's missing
- B-tasks **>30%** of work time → surface automation/delegation candidates
- Pipeline failures **cluster at one stage** → that's the dominant bottleneck
- Encoding failures repeat on same project → cholinergic stack signal
- E-task time vs revenue: flag when epistemic work crowds out revenue work

## Do NOT Surface:

- Speculative correlations without repeated observation
- Suggestions to add more tracking variables
- Prompts to reflect beyond 1-2 word task notes
- Analysis as substitute for execution — if priority task not started, offer the task
