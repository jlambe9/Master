# Intervention Drivers — Default Menus

*Loaded when a B/A task needs intervention. Refined through use.*
*Protocol + conflict table: see `intervention-library.md`.*
*Status starts UNTESTED. Updates from execution logs in `task-log.md`.*

## Zero Epistemic Value (🟡 B — "empty" feeling)

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Music (instrumental) | Auditory SEEKING without cognitive load | No verbal processing needed | Reading/calls |
| Body doubling | Social PLAY/CARE as background engagement | Desk tasks | Adds pressure (see conflicts) |
| Cardio before (15 min) | DA/NE boost lasts 60-90 min | Before planned B-blocks | Already post-gym |
| Stimulant timing | Peak effect during B-block | Timing controllable | Already dosed |
| Micro-epistemic injection | Embed genuine question in task | Real question exists | Fake questions don't work |
| Time-box 25 min | Constraint = minimal structure | All B-tasks (non-negotiable) | Never skip |
| Batch similar B-tasks | Reduce context-switching | Multiple B-tasks same day | Total >60 min |

## High Interpersonal Risk (🔴 A — "judgment/rejection" feeling)

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Preparation script | Reduces ambiguity, gives PFC procedure | Cold outreach, calls, negotiations | Casual interactions |
| Outcome pre-processing | Bounds worst case, deflates infinite-negative | High-stakes calls, pitches | Truly catastrophic outcomes (rare) |
| Smallest first action | Builds approach momentum, disconfirms threat | Initiation is failure point | First action IS the aversive part |
| Body doubling (accountability) | External precision overrides avoidance | "Need to do this" aversion | "Don't want to be seen" aversion |
| Morning scheduling | More PFC = more override capacity | All A-I tasks | Time-sensitive, can't wait |

## High Unpredictability (🔴 A — "don't know how long this traps me")

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Hard time limit + exit strategy | Bounds worst case. Exit must be specific + pre-decided. | Calls, bureaucratic interactions | Abandoning creates worse problems |
| Parallel E-task during wait | Converts dead time to productive time | Hold music, waiting rooms | Full attention tasks |
| Delegate waiting component | VA handles hold/navigation, transfers at decision point | Known long holds (HMRC, insurance) | Identity verification required |
| Information gathering first | Reduce unknowns before committing (E-task) | All high-unpredictability tasks | Don't use as delay tactic |

## High Confrontation (🔴 A — "someone might be hostile")

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Written channel | Time to compose, paper trail, draft-review-send | Disputes, complaints, negotiations | Written already failed |
| Preparation script + branching | Pre-decide responses to provocation | Potentially hostile calls/meetings | Over-engineering casual interactions |
| Advocate/intermediary | Professional handles adversarial interaction | Tax, legal, formal complaints | Personal presence required |
| Post-interaction debrief (30 sec) | Updates priors — without it emotional memory dominates | Always after confrontation | Never skip |

## High Identity Relevance (🔴 A — "this reflects on who I am")

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Draft-first, ship-later | Separate creation (E) from publishing (A-I) | Content, proposals, applications | Inseparable (live presentations) |
| Minimum viable version | "Worst version I'd ship?" B+ shipped > A+ unshipped | Publishing, launches | Quality genuinely matters (rare) |
| De-couple identity from output | Surface and reality-check the identity threat | Recurring avoidance | First attempt (try faster interventions first) |

## High Exposure / Visibility (🔴 A — "people will see me")

| Intervention | Mechanism | Use When | Skip When |
|-------------|-----------|----------|-----------|
| Graduated visibility | DM 1 person → small group → public. Each step disconfirms. | Content, speaking, community | Can't be graduated (keynote) |
| Audience reframe | "Who specifically am I imagining? What would they actually think?" | Before publishing. 60 seconds. | Concern is legitimate |
