## Working Backwards

```
£X/month needed to survive (from FIN-003)
÷ £500/client = X clients/month
÷ 25% close rate = X calls/month
÷ 70% show rate = X calls booked/month
÷ 25% convo→call = X conversations/month
÷ 25% reply→convo = X replies/month
÷ 10% intro→reply = X intros sent/month
÷ 42.9% accept rate = X CRs sent/month
÷ 30 days = X CRs/day
```

4. Book X calls this week
5. [Whatever the maths says]

This becomes the morning briefing priority list — not aspirational, survival-driven.
